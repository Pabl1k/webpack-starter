# WEBPACK-STARTER
It's an empty React project to build from scratch.

## About the project
By default, project mode set to Development (port: 8080).
To change mode to Production (port: 5000), in terminal type: 
```mode
cd .\build
```
then
```mode
npx serve
```

## Installation
Run 
```install
npm install
```
To start the project
```start
npm start
```
